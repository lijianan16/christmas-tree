====================================
  魔法圣诞树 - 使用说明
====================================

【方式一：使用启动脚本（推荐）】

Windows 用户：
1. 双击 "启动服务器.bat" 文件
2. 等待浏览器自动打开页面
3. 如果提示需要安装 Node.js，请访问 https://nodejs.org/ 下载安装

Mac/Linux 用户：
1. 右键点击 "启动服务器.sh"，选择"打开终端运行"
2. 或者在终端中执行：chmod +x 启动服务器.sh && ./启动服务器.sh
3. 等待浏览器自动打开页面

【方式二：手动启动（如果脚本不工作）】

1. 确保已安装 Node.js（访问 https://nodejs.org/ 下载）
2. 打开终端/命令提示符
3. 进入此文件夹
4. 执行命令：npx http-server -p 8080
5. 在浏览器中访问：http://127.0.0.1:8080/ai_studio_code.html

【方式三：在线访问（最简单，无需安装）】

如果代码已部署到 GitHub Pages 或其他在线服务：
直接访问提供的网址即可，无需任何安装。

【功能说明】

键盘控制：
- 1 = 树形模式
- 2 = 散开模式  
- 3 = 聚焦模式
- 4 = 心型模式（红色实心）
- F = 全屏显示
- H = 隐藏/显示控制界面

手势控制（需要摄像头）：
- 握拳 = 树形模式
- 张开手掌 = 散开模式
- 捏合（拇指+食指）= 聚焦模式
- 比心（食指+中指V字）= 心型模式
- 移动手掌 = 旋转场景

【注意事项】

1. 必须通过 HTTP 服务器访问，不能直接双击 HTML 文件
2. 需要浏览器支持 WebGL（Chrome、Firefox、Edge 都可以）
3. 手势识别需要摄像头权限
4. 如果摄像头无法使用，可以使用键盘控制

【常见问题】

Q: 提示"未检测到 Node.js"？
A: 请访问 https://nodejs.org/ 下载并安装 Node.js（选择 LTS 版本）

Q: 页面一直加载？
A: 可能是网络问题，等待 15 秒会自动使用降级模式（键盘控制）

Q: 摄像头无法使用？
A: 点击"启用摄像头"按钮，或在浏览器设置中允许摄像头权限

Q: 如何退出全屏？
A: 按 F 键或 ESC 键，或点击"退出全屏"按钮
